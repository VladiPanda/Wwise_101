Invalid float in parameter ... of function ...

The specified float parameter is not a number (NAN) or an infinite (+/- INF), neither of which is allowed as an input parameter in any function of the sound engine.

Recommended resolution steps:

  • Check all calls to the reported function from the game code. Validate the input parameters.

  • Enable API Calls in the Profiler Settings and reproduce the scenario. Look for new errors indicating invalid values on API calls.
