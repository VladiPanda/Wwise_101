Selected media ... not available.

A sound was selected to play but its media is not loaded. The ID of the sound and the media file are reported on the same line.

Probable causes:

  • The sound structure and the sound media are in two different SoundBanks, and only the structure is loaded.

  • The sound structure and the sound media are in two different SoundBanks, and they were not generated with the same content following a media file change.

  • Some SoundBanks from older builds were loaded, with out-of-date data.

Recommended resolution steps:

  • Find which SoundBank contains the desired media file and make sure it is loaded earlier.

  • Make sure the SoundBanks are up to date by regenerating them.

  • Make sure all SoundBanks are properly deployed on your test console.
