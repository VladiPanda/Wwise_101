Missing music node parent. Make sure all banks containing music structures are completely loaded.

This error message indicates a situation where the user can add a transition segment that is not part of the same music tree, then explicitly exclude the parent of the transition segment from the bank.

Recommended resolution steps:

  • Include the parent in the music SoundBank(s).

  • Load the missing music SoundBank, if that is the case.

  • Re-parent the transition segment in the same tree.
