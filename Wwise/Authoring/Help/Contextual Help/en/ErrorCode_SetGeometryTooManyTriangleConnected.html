AK::SpatialAudio::SetGeometry - More than two triangles are connected to the same edge.

One of the triangles provided in AK::SpatialAudio::SetGeometry is not properly constructed. The API requires that only two triangles share the same edge.

Recommended resolution steps:

  • Profile the API calls the game is making by enabling API Calls in the Profiler Settings (Alt-G). Retest your scenario and check the parameters given to the function prior to this error.

    The error string indicates the indices in the Triangles array of the three triangles that erroneously share the same edge, and the indices in the Vertices array that form said edge.
