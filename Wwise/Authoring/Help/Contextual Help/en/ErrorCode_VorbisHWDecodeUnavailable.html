Vorbis Hardware Acceleration library not found. Source will be decoded in software.

The sound engine was initialized with the Vorbis Hardware Acceleration option enabled, but the required PRX library was not loaded.

Until the library is loaded, every Vorbis source will issue this error once and will then be decoded on the CPU.

Probable causes:

  • The AK::LoadVorbisHwAcceleratorLibrary API function was not called by the game.

  • The PRX library was not packaged properly with the game.

Recommended resolution step:

  • Ensure that the game properly loads the Vorbis Hardware Acceleration library using the AK::LoadVorbisHwAcceleratorLibrary API call prior to playing any Vorbis source. The function only needs to be called once, and it can be called prior to sound engine initialization. Make sure the function call returns successfully.
