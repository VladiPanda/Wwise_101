Not a valid Switch value ... in group ...

Indicates that a Switch Container was triggered to play but could not play any sound.

Probable causes:

  • In your project, there is no sounds associated with the Switch Values in the Switch Container in the Assigned Objects window.

  • In your project, there is no "Default Switch/State" defined in the properties of the Switch Container and no Switch Value was set on the Game Object in your game.

  • In your game, a Switch Value was set on the game object, but there is no Assigned Object associated with it in the "Assigned Object" section.
