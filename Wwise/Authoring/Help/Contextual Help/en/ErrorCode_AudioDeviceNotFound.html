AkOutputDeviceID not found as an active device in call to function ...

This error occurs when Wwise cannot find a device in use by the sound engine that matches the provided AkOutputDeviceID.

Recommended resolution steps:

  • Retry the same scenario with a Debug version, with a debugger attached to the game.

  • Check that the AkOutputDeviceID being provided for replacement matches the AkOutputDeviceID used during AK::SoundEngine::Init. Note that if the audioDeviceShareset in the provided AkInitSettings::settingsMainOutput is 0, then the initial AkOutputDeviceID will be the ID of a ShareSet named "System" mixed with the idDevice from the provided AkInitSettings::settingsMainOutput.

  • Check that the AkOutputDeviceID being provided for replacement matches an AkOutputDeviceID previously returned from AK::SoundEngine::AddOutput() or AK::SoundEngine::ReplaceOutput().

  • Check that a call to AK::SoundEngine::RemoveOutput() with the AkOutputDeviceID in question was not made.
