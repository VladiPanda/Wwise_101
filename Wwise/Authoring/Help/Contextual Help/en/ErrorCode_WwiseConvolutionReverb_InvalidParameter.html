Convolution Reverb plug-in parameters are invalid, effect is disabled.

This error occurs if one or more Convolution Reverb parameters are invalid.

Possible causes:

  • The SoundBank loaded was not generated for the current platform. This might occur when the same SoundBank is used for all platforms.

Recommended resolution steps:

  • Verify the Convolution Reverb parameters in the Wwise project.

  • Regenerate the SoundBank(s) containing the plug-in.
