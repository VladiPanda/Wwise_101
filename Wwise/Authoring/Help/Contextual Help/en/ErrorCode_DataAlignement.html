Bank or Media pointer not aligned to ... bytes.

This error occurs when using either AK::SoundEngine::LoadBank (memory pointer version) or AK::SoundEngine::SetMedia and the data pointer passed is not aligned to the expected granularity. This varies by platform and by codec. Default requirement is 16 bytes.

Probable causes:

  • The media pointer is not aligned to the default alignment of 16 bytes.

  • ATRAC9 is used and requires an alignment of 256 bytes.

  • OpusNX is used and requires an alignment of 16 bytes.

Recommended resolution steps:

  • Allocate memory with the proper alignment prior to loading the data.
