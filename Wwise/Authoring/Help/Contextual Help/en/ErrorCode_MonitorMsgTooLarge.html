Monitor Queue message is too large. Increase the size of uMonitorQueuePoolSize. Message size ... bytes.

The Monitor Queue, used by the Wwise Profiler, could not contain one of the messages to send to the Profiler. The maximum size of a message is half of AkInitSettings::uMonitorQueuePoolSize. This error can be triggered when profiling the API calls, as certain APIs can pass unbounded arrays of parameters. The only side effect of this error is a missing message in the Wwise Profiler.

Recommended resolution steps:

  • Verify calls to AK::SoundEngine::SetMultiplePositions and all Spatial Audio geometry functions to reduce the counts of the parameter arrays. This message is often a sign of a badly formed API call.

  • Increase AkInitSettings::uMonitorQueuePoolSize
