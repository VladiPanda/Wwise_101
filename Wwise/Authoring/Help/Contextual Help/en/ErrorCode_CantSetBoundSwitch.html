Switch ... is bound to RTPC ... It can not be set directly.

This error occurs when a Switch value is being explicitly set through an Event or a function call, but this Switch Group is driven by an RTPC. The Switch value will be ignored.

Recommended resolution steps:

  • Avoid using that Switch Group in a Set Switch Action in an Event.

  • Avoid using that Switch Group in a SetSwitch function call.

  • Deselect the Use Game Parameter option in the Switch Group definition in the Wwise Project.
