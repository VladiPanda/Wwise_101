External source missing from PostEvent call.

A call to AK::SoundEngine::PostEvent was made with an Event that requires one or more External Source file replacements. Not all required sources were provided with the API call. Note that it is possible to have multiple External Sources in a single Event and that all of them must be provided in code at runtime. The External Source ID must match as well.

Recommended resolution steps:

  • In the Wwise project, verify what is the target of the Event and how many External Sources are needed in that Event. Take note of either the string or numeric identifier. These IDs must be provided to PostEvent. In the call to PostEvent, make sure the same number of External Sources is present in the array, and that their IDs match the ones in the Wwise project.
