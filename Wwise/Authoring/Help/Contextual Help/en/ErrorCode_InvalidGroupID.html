Invalid State Group ID.

The State Group name or numeric ID used with an AK::SoundEngine::SetState call is not found.

Probable causes:

  • The Init SoundBank is not loaded yet or could not load properly.

  • The Init SoundBank is out of date and doesn't contain the latest changes.

  • There is a typo in the State Group name or ID in the code.

Recommended resolution steps:

  • Verify that the Init SoundBank is loaded by checking the Wwise Profiler's SoundBanks tab.

  • Regenerate the SoundBanks and redeploy to the target console.

  • Verify the name of the State Group.
