Not enough memory in I/O pool to start stream.

The I/O pool doesn't have enough memory to read the header of a streamed file. The sound will be terminated.

Recommended resolution steps:

  • Increase the amount of memory in the I/O pool. Set AkDeviceSettings::uIOMemorySizebefore initializing the Streaming Manager in your initialization sequence.

  • Connect the Wwise profiler and check the Streaming tab. Validate that there isn't an abnormal number of streamed voices.

  • Play less streaming sounds at the same time using virtual voices and limits.

Recommended resolution steps:

  • Managing Low-Volume Objects

  • Limiting Object Playback Instances
