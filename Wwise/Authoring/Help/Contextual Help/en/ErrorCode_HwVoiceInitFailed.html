Failed creating hardware-accelerated voice.

This error occurs when a sound using a hardware-accelerated audio format is unable to acquire the proper resources from the platform's operating system in order to start decoding. This sound will not play.

Possible causes:

  • A corrupted bank or media could cause this issue.

  • Exceeding the number of voices or memory reserved for hardware operations.

Recommended resolution steps:

  • Regenerate your SoundBanks and media. If the problem persists, it might be caused by faulty hardware, a bug in the game, or Wwise itself. In this case, contact Audiokinetic Support.
