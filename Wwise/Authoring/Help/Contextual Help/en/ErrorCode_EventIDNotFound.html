Event ID not found.

PostEvent was called but the sound engine could not find the specified Event. The Event used will be reported on the same line.

Probable causes:

  • A SoundBank was not loaded or didn't load properly.

  • SoundBanks are out of date.

  • Typo in the Event name.

  • Typo in the Event ID.

Recommended resolution steps:

  • Make sure the name or ID is correctly written in the game code.

  • Check for SoundBank loading errors prior to this error and fix them.

  • Ensure the Event is in a SoundBank and that the SoundBank is loaded by checking the SoundBanks tab in the Advanced Profiler.

  • Make sure the SoundBanks are up to date.

  • Make sure the SoundBanks are properly packaged with the game during the build process.

  • Make sure the SoundBanks are properly deployed on the target console/mobile.
