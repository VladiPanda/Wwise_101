Plug-in unsupported channel configuration.

This error means the listed plug-in (source, effect or mixer) doesn't support the channel configuration of the object it is applied on.

Probable causes:

  • The Bus Channel Configuration property is incompatible with the Mixer or Effect plug-in. This can be the case for ambisonic specific plug-ins. Other plug-ins might not support other channel configurations.

Recommended resolution steps:

  • Check the documentation of the specific plug-in that is failing.
