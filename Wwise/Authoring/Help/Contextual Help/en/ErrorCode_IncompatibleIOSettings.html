I/O settings incompatible. Block size must be between one and Granularity and whole fraction of the granularity.

The I/O settings given in AkDeviceSettings are not consistent or invalid.

Probable causes:

  • The Block Size must be between one and the Granularity setting. The Granularity setting must be a whole multiple of the Block Size.

  • uBufferSize is not aligned with the platform's buffer alignment (usually 16 bytes)

Recommended resolution steps:

  • Review the AkDeviceSettings used when calling AK::StreamMgr::Create.

  • Compile the game in Debug and rerun the scenario. Report any asserts to Audiokinetic Support.

  • If your game implements its own version of the Stream Manager or its own I/O hook, attach the debugger and trace the error.
