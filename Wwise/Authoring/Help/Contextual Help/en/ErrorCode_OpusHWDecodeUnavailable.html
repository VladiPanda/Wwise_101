AkInitSettings::uNumSamplesPerFrame value unsupported by Opus hardware decoder. Supported size: 256, 512 and 1024. Opus will be disabled.

The hardware Opus decoder can only work with the three buffer sizes listed. All sounds using Opus will print an error and fail to play.

Recommended resolution steps:

  • Change AkInitSettings::uNumSamplesPerFrame to 256, 512, or 1024, before the call to AK::SoundEngine::Init.
