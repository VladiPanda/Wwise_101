Lower engine command list is full.

This error indicates that an internal process ran out of memory. It is always preceded by an Insufficient Memory error.

Recommended resolution steps:

  • Increase the memory limit by changing AkMemSettings.uMemAllocationSizeLimit before calling AK::SoundEngine::Init. See Advanced Sound Engine Integration for more information on initialization settings.

  • Better manage your memory by killing voices using Playback Limits and Virtual Voices.
