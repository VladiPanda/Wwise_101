Music engine not initialized : Content cannot be loaded from bank.

The Interactive Music engine ( AkMusicEngine ) was not initialized, or failed to initialize, before the SoundBanks were loaded.

Probable causes:

  • AK::MusicEngine::Init failed. Check the return code of this function in your game initializing code.

  • The Interactive Music feature was not integrated in your game. It is an option. Read more here: Integrating Interactive Music
