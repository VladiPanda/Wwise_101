Playing a trigger-rate container as part of a sample-accurate playlist is not supported and will sound out-of-sync.

This error refers to a Sequence Container or a Random Container that has its Transition Type set to Trigger rate, and is itself a child of a Sequence Container or a Random Container that has its Transition Type set to Sample accurate.

This specific combination is simply not supported by Wwise. The Trigger rate container will only play one sound and then silence until the next frame.

Recommended resolution steps:

  • Change the structure or properties to avoid this container setup.

  • If you are trying to do grain synthesis, try the SoundSeed Grain plug-in.
