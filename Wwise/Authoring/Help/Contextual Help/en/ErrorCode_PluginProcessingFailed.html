Plug-in execution failure.

This error means the listed plug-in (source, effect, mixer or sink) failed at runtime, while processing audio data.

Probable causes:

  • Usually happens because there is not enough memory to allocate temporary processing buffers.

  • Some third-party plug-ins might generate this error for internal reasons.

Recommended resolution steps:

  • Check for memory errors appearing before this "Plug-in execution failure" error. More total memory may be required.
