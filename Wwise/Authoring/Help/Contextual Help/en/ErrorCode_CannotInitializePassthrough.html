Cannot initialize passthrough. Passthrough and objects will be disabled.

This is an error that can occur if the 3D object processor could not be initialized. The object's audio will be mixed to the main mix instead.

Probable causes:

  • The 3D Audio processing library or hardware is in use by another process (usually Unreal or Unity).

Recommended resolution steps:

  • Make sure Unreal or Unity's audio system is disabled.
