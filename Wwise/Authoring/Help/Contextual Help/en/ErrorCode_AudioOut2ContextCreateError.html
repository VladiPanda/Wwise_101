sceAudioOut2ContextCreate Failed.

A fatal error occurred while initializing the PlayStation's core audio output system. An error code of -2144960510, or SCE_AUDIO_OUT2_ERROR_OUT_OF_RESOURCE (0x80268002) is typically returned. This type of error can occur if Wwise attempts to initialize the sceAudioOut2 Context with one or more Object Ports (required for 3D Audio) after another system or sound engine in the application has already created an sceAudioOut2 Context with some Object Ports: the PS5 only supports initialization of one sceAudioOut2 Context with Object Ports. To avoid this error, either initialize Wwise with zero Object Ports, or disable the other sound engine's 3D Audio support. If other error codes are returned or if this error continues to occur even if Wwise is initialized without Object Ports, contact Audiokinetic Support.
