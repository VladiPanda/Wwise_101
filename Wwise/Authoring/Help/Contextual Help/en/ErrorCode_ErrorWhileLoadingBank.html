Error while loading bank.

This error that indicates the SoundBank loading process failed while decoding the SoundBank structures. This is a last resort, catch-all error. Check for other errors preceding this one in the Capture Log.

Probable causes:

  • A SoundBank is corrupted on disk.

  • An unhandled insufficient-memory error occurred. It is a bug that should be reported.

  • An unforeseen error.

Recommended resolution steps:

  • Check for out-of-memory errors in the log and resolve them.

  • Regenerate your SoundBanks and redeploy to the target console.

  • Contact Audiokinetic Support. Ideally, capture a ProfilingSession.prof while reproducing the problem. (Refer to ??? for details.)
