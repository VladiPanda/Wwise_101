Media updated: ... transferred from Wwise Project. Glitches might be heard. Previous media from other sources (bank, prepared, etc) will be ignored.

The specified media file was added or modified in the Wwise project. When Wwise is connected to a game, it updates the game with new or modified media. This media update is not permanent; the files on disk are not modified. Any version of that media that was already in memory through bank loads will be ignored.

This message is also an indication that the SoundBanks are not up to date in the deployed game's build. It's not necessary to update them immediately, but the game will sound different when Wwise is connected compared to when it's not until the changes are deployed.

This could be triggered by:

  • A new file in a new Sound or Voice in an existing Container.

  • A new file replacing an existing file as a Sound, Voice, or Music Track source.

  • Changing the Begin or End markers, or any of the other Source properties, in the Source Editor.

  • Changing the Conversion Settings of a Sound, Voice, or Music Track.

  • Changing the Impulse Response of an AK Convolution Reverb.

  • Adding a new AK Convolution Reverb.

  • Changing the plug-in specific media of any other source or Effect plug-in.

Note

If you do not want Wwise to update media in the game, select Profile Only from the list in the Remote Connections dialog when connecting to your game.
