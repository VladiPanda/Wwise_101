Could not register plugin.

The specified Dynamic Library for a plug-in was found on disk but there was an error while loading the library. This is a typical symptom of external dependencies of the plug-in not being present at load time.

Recommended resolution steps:

  • If this is a third party plug-in (not made by Audiokinetic), read the documentation about the plug-in to validate prerequisites.

  • Contact the plug-in vendor for support.
