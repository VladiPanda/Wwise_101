Source file is of different format than expected.

This error occurs when a streamed file or an external source isn't using the same codec as noted in the associated structural SoundBank or AK::SoundEngine::PostEvent() call.

Probable causes:

  • A recent change in codecs in the Conversion Settings of the project.

  • For External Sources, a mismatch between the codec specified in the format passed in the AkExternalSourceInfo structure for the AK::SoundEngine::PostEvent() call and the actual file format.

Recommended resolution steps:

  • Make sure all SoundBanks and WEM files (streamed files) are generated and properly deployed on the target console.

  • If using External Source, validate the codec used in the specified Conversion Settings in the WSOURCES file.
