The game object for an event was not registered as an emitter. Make sure the game object is registered as an emitter before using it to post an event.

This error can occur as a followup to a previous error, such as an Insufficient Memory error. It means that the Game Object does exist, but its initialization was incomplete.

Recommended resolution steps:

  • Inspect the Capture Log for other errors pertaining to the same Game Object and correct those errors first.

  • Contact Audiokinetic Support if the error persists.
