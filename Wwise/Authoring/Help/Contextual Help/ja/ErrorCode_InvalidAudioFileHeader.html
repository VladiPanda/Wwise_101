Invalid file header.

ストリーミングファイルのファイルヘッダが破損しているため、または対応していないファイルタイプのため、デコードできなかったことを示します。このエラーが発生したサウンドもエラーと共に報告されます。

考えられる原因:

  • ファイルの破損。

  • 誤ったバージョンのWwiseがファイルの変換に使用されました。

  • AkExternalSourceでWEM以外のフォーマットのファイルを使用しました。

推奨される解決策:

  • ファイルを再変換して再コピーします。

  • ゲームで使用するサウンドエンジンに対応するWwiseバージョンを使用します。

  • AkExternalSourceを使用する時はWEMファイルを使用します。

  • Audiokineticサポートが分析するため、オリジナルファイルと変換後のファイルをAudiokineticサポートに送ります。
