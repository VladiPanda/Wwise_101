Plug-in not found.

ゲームが不明のソース、エフェクト、ミキサー、またはシンクプラグインを使い、SoundBankをロードしたことを意味するエラーです。このプラグインを実行するためのコードがみつかりません。

考えられる原因:

  • The game code doesn't static link with the plug-in library. この問題は、すべてのUnrealゲームと、多くのインハウスエンジンで、発生する可能性があります。

  • The game cannot find the plug-in dynamic library (DLL or SO). この問題は、すべてのUnityゲームと、一部のインハウスエンジンで、発生する可能性があります。

  • The game cannot load the plug-in dynamic library (DLL or SO) because of a missing dependency. プラグインの関連ドキュメントで、要件を確認してください。

  • SoundBankが最新のものではありません。

  • サウンドバンクが破損しています。

  • Init.bnkまたは AK::SoundEngine::RegisterPlugin の明示的な呼び出しによってロードされていないプラグインに対して、AK::SoundEngine::GetDeviceList をコールしました。

推奨される解決策:

  • あなたのゲームとWwiseライブラリを、スタティックリンクでつなげる場合は、ゲームの#include "ThePluginFactory.h" や、リンクオプションを確認します。さらに詳しく: 統合の詳細 - プラグイン

  • あなたのゲームを（Unityなどと）、ダイナミックリンクでつなげる場合は、SoundBanksを再生成し、ゲームをビルドします。

  • ゲームのSoundBanksを、再生成し再パッケージします（Init.bnkを含む）。

  • シンクプラグインの場合、このプラグインのShareSetが最低1つのマスターバスに関連付けられていることを確認します。
