Audio Device not found in Init bank: ...

APIコールが参照したAudio Deviceが見つかりませんでした。

考えられる原因:

  • Initバンクがまだロードされていません。

  • 関数パラメータで渡された名前に誤字がありました。

  • Audio Deviceをプロジェクトに追加した後にInitバンクを再生成していなかった、または再展開していませんでした。

推奨される解決策:

  • Wwise Profilerを接続し、このエラーの前にInitバンクがロードされていることを確認します。

  • Profiler Settings （Alt-G）で API Calls を有効にします。Wwise Profilerを接続し、このエラーをトリガーする関数に渡しているパラメータを検証します。

  • テストコンソールでバンクを再生成して再展開します。
