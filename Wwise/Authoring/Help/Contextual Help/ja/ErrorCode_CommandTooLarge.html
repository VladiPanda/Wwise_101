Audio command is too large to fit in the command queue. Break the command in smaller pieces.

このエラーは、Wwise SDK APIへの1回のファンクションコールが、コマンドキュー全体に対して大きすぎる場合に発生します。このエラーをトリガーする可能性があるのが、パラメータとしてアレイを送信するファンクションです。頻繁に問題を引き起こすのが AK::SoundEngine::SetMultiplePositionsで、1回のコールで過度のポジション数（数千）が含まれてしまうことがあります。アレイを小さいものに分け、複数のファンクションコールを実行させてください。
