Cross-Fade transition ignored: Estimated length not available. Make sure sounds with Cross-Fade transitions virtual mode are set to "Play from elapsed time" and that only sounds with finite duration are used.

クロスフェードのパラメータを計算するには、サウンドの本当の長さが必要です。定義上、無限のサウンドは終了しないので、そこからクロスフェードすることはできません。サウンドのVirtual Mode設定では、Play From BeginningやResumeのオプションを使えば、サウンドが実際のメディアの長さよりも格段と長く続くことが考えられるので、サウンドの実際の長さを左右する設定です。

クロスフェードは、"from"と"to"のどちらのサウンドでも起きません。

推奨される解決策:

  • Validate the Virtual Voice Behavior in the Property Editor used for the sound associated with this error. 使ってよいのは、"Play from Elapsed"と、"Continue To Play"だけです。

  • 音が無限にループしていないか、確認します。
