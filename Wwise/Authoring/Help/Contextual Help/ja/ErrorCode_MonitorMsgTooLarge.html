Monitor Queue message is too large. Increase the size of uMonitorQueuePoolSize. Message size ... bytes.

Wwise Profilerの使用するMonitor Queueが、Profilerに送信するメッセージの1つを受け付けられませんでした。メッセージサイズの上限は AkInitSettings::uMonitorQueuePoolSize の半分です。このエラーはAPIコールのプロファイリング中にトリガーされる可能性があります。これは特定のAPIがパラメータの配列を無制限に渡すことができることが原因です。このエラーによる唯一の副作用は、メッセージがWwise Profilerに表示されないことです。

推奨される解決策:

  • AK::SoundEngine::SetMultiplePositions およびすべてのSpatial Audioジオメトリ関数のコールを確認し、パラメータ配列の数を減らします。このメッセージは多くの場合、不正な形式のAPIコールを示しています。

  • AkInitSettings::uMonitorQueuePoolSize を増やします
