Type mismatch while loading bank. Object ... is a ... in the currently loading bank. It was a ... in bank ...

このエラーはWwiseで使用する固有の識別子の衝突を示します。2つのオブジェクトが2つの異なるサウンドバンクにあり、同じ識別子を共有しています。

考えられる原因:

  • 通常コンソールのテストディレクトリには、古くて使われていないサウンドバンクが存在します。

  • 識別子のコリジョンも実際にありますが、非常にまれです。

推奨される解決策:

  • テストコンソールからすべての.bnkファイルを削除します。

  • 報告されたオブジェクトの名前をWwise Projectで変更し、サウンドバンクを再生成します。

  • この問題は内部バグの可能性もあるため、Audiokineticサポートにご連絡ください。
