Invalid transform in function ...

指定された関数に無効な方向が渡されました。

考えられる原因:

  • AkTransform のオリエンテーションベクトルのFrontとTopが、互いに直交しません。

  • AkTransform のオリエンテーションベクトルのFrontとTopの1つ、または両方の長さが、1に等しくありません。

推奨される解決策:

  • API Calls を Profiler Settings （Alt-G）で有効にしてゲームが行っているAPIコールのプロファイリングを行います。シナリオを再テストして、このエラーが発生する前に関数に送られたパラメータを確認します。
