AK::SpatialAudio::SetGeometry - Triangle number ... is skipped because it is invalid; two or more of its vertices are at the same position.

AK::SpatialAudio::SetGeometry に提供した三角形が正しく定義されていません。

考えられる原因:

  • この問題は同じ座標の頂点を同じ三角形で使用し、線または点のみを定義したことが原因です。

推奨される解決策:

  • API Calls を Profiler Settings （Alt-G）で有効にしてゲームが行っているAPIコールのプロファイリングを行います。シナリオを再テストして、このエラーが発生する前に関数に送られたパラメータを確認します。

    エラーメッセージの数字はTriangle配列の三角形インデックスと、それに対応するVertex配列の頂点インデックスを示します。
