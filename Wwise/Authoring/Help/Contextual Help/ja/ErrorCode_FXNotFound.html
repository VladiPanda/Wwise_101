Effect ShareSet not found in function ...

指定された関数のパラメータで参照されるEffect ShareSetが、その時点でロードされているサウンドバンクの中に見つかりませんでした。

考えられる原因:

  • このEffect ShareSetの入ったサウンドバンクがまだロードされていないか、このコールの前にアンロードされました。

  • このEffect ShareSetがプロジェクトのどのサウンドバンクにも入っていません。

  • 関数パラメータで渡されたIDに誤字がありました。

  • Effect ShareSetをプロジェクトに追加してからサウンドバンクを再生成していなかった、または再展開していませんでした。

注釈

Effect ShareSets that are used by Busses are packaged in the Init SoundBank if the Bus leads to a used Audio Device ShareSet. If not, the Effect ShareSet must be manually added to a User-Defined SoundBank, and that SoundBank must be loaded by the game.

推奨される解決策:

  • Wwise Profilerを接続し、このエラーが発生する前にこのShareSetを含むサウンドバンクがロードされているかどうかを確認します。

  • Profiler Settings で API Calls を有効にします。Wwise Profilerを接続し、このエラーをトリガーする関数に渡しているパラメータを検証します。または、ゲームコード内の指定された関数コールにブレークポイントを設定します。

  • テストコンソールでバンクを再生成して再展開します。
