Selected node not available. Make sure the structure associated to the event is loaded or that the event has been prepared.

再生するために選択したサウンドが、ロードされていないことを示します。このサウンドのIDが同じラインに表示されます。

考えられる原因:

  • イベントが、ロードされていないサウンドを参照しています。

  • Random Containerが、ロードされていない子サウンドをスタートさせようとしています。

  • Switch Containerが、ロードされていない子サウンドをスタートさせようとしています。

  • ミュージックプレイリストが、ロードされていない子サウンドをスタートさせようとしています。

  • ダイナミックイベントが、ロードされていない子サウンドをスタートさせようとしています。

  • SoundBankが最新のものでないため、選択したサウンドIDが無効です。

推奨される解決策:

  • Capture Logを見て、このエラーを選択する。関連するノティフィケーション（関連するドットインジケータ）を追い、エラーの原因となったEventを探します。

  • サウンドがサウンドバンクに入っているのか、そのサウンドバンクをロードしたのか、を確認する。

  • SoundBankが最新状態にあることを確認する。

  • サウンドバンクがテストコンソールに正しく展開されるようにします。
