AK::SpatialAudio::SetGeometry - Triangle formed by vertices [v0, v1, v2] is too large.

AK::SpatialAudio::SetGeometry に提供した三角形が大きすぎます。

考えられる原因:

  • この問題は三角形の2つ以上の頂点が離れすぎているため、特定の演算を実行したときに浮動小数点オーバーフローが発生してしまうことが原因です。

推奨される解決策:

  • API Calls を Profiler Settings （Alt-G）で有効にしてゲームが行っているAPIコールのプロファイリングを行います。シナリオを再テストして、このエラーが発生する前に関数に送られたパラメータを確認します。

    エラーメッセージの数字は大きすぎる三角形のTriangle配列の三角形インデックスと、それに対応するVertex配列の頂点インデックスを表します。
