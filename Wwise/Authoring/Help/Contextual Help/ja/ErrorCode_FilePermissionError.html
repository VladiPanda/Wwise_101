File ... could not be opened due to permissions, access rights or conflicting open modes. (Found in path(s): ...).

考えられる原因:

  • The file was opened in exclusive mode by a different application.

  • The file is in an access-protected directory.

  • An attempt was made to read a recording being made by the Recorder Effect plug-in without first stopping the recording process.

推奨される解決策:

  • Check your user rights in the specified folders.

  • Check the settings of your version control system for the reported file.

  • Check the anti-virus settings on your computer.

  • If related to an audio recording made with the Recorder Effect plug-in, make sure the recording is stopped.
