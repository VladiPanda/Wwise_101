AK::SpatialAudio::SetPortal: Portal must have a front room which is distinct from its back room.

Portalは2つの異なるRoomを接続する必要があります。 AK::SpatialAudio::SetPortal に対し、 AkPortalParams::FrontRoom や AkPortalParams::BackRoom 経由で渡されたRoom IDは、同じものです。

推奨される解決策:

  • API Calls を Profiler Settings （Alt-G）で有効にしてゲームが行っているAPIコールのプロファイリングを行います。シナリオを再テストして、このエラーが発生する前に関数に送られたパラメータを確認します。

  • AK::SpatialAudio::SetPortal に渡される AkPortalParams::FrontRoom と AkPortalParams::BackRoom で指定したRoomは別である必要があります。
