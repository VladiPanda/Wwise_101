File ... not found in path(s): ...

指定のファイルが見つかりませんでした。検索したパスの一覧が表示されます。

考えられる原因:

  • サウンドバンクのパスが設定されていないか、正しいフォルダが設定されていません。

  • ファイルがテストコンソールに正しく展開されませんでした。

  • このファイルはWwise Projectによって生成されたものではありません。

  • ファイルは展開パッケージまたはバンドルにパッケージ化されていません。

推奨される解決策:

  • Wwiseが提供するデフォルトのI/Oコードを使用する場合は、 CAkDefaultIOHookBlocking::SetBasePath （または CAkDefaultIOHookBlocking::AddBasePath ）へのコールが適切なディレクトリを指していることを確認します。

  • サウンドバンクまたはWEMファイルが適切に生成されていることを確認します。

  • サウンドバンクが適切にパッケージされターゲットに展開されていることを確認します。

  • Unityを使用する場合はゲームをパッケージ化する前に、ファイルがUnityプロジェクトのStreamedAssetsディレクトリのどこかに存在していることを確認してください。

  • カスタムI/Oフックを使用している場合は、カスタムI/Oフックのコードをデバッグしてください。
