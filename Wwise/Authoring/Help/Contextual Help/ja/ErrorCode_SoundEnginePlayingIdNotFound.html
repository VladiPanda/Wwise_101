Playing ID ... not found in function ...

指定された関数を実行するには、有効な再生中IDパラメータが必要です。提供された識別子はサウンドエンジンにとって未知のものです。

再生中のIDを返す関数は AK::SoundEngine::PostEvent() です。

考えられる原因:

  • 提供された再生中IDは完全に終了したか、停止されたEventに属します。

  • 再生中IDが AK::SoundEngine::PostEvent() によって返されませんでした。

推奨される解決策:

  • ゲームコードで指定の関数コールを見つけ、Playing ID値の出所をデバッグします。

  • Wwise Profilerを使用してこのコール前にイベントが停止していたかどうかを確認します。
