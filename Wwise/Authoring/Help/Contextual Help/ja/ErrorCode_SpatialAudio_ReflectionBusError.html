The playing sound is assigned the same Early Reflection Auxiliary Bus in the Authoring Tool that has been set via AK::SpatialAudio::SetImageSource. Use a unique bus to avoid image source conflicts.

Wwise Authoringのサウンドに割り当てたEarly Reflections Auxiliary Busが、 AK::SpatialAudio::SetImageSource に渡したバスと競合します。

考えられる原因:

  • Wwise Authoringのサウンドに割り当てられたEarly Reflections Auxiliary Busが、同じバスを持ち、API関数 AK::SpatialAudio::SetImageSource に渡された同じゲームオブジェクトで再生されています。Reflectの単一のインスタンスはジオメトリAPIとSetImageSource APIの両方からイメージソースデータを受け取ることはできません。

推奨される解決策:

  • ジオメトリリフレクションAPIとSetImageSource APIを併用する場合は、同じゲームオブジェクトで再生されるサウンドに対してWwise Authoringで定義したAuxiliary Busとは別のAUXバスをSetImageSourceに指定します。これはReflectプラグインを含む目的のAUXバスの複製を作成し、それを競合するサウンドに割り当てることで実現できます。こうすることで、このサウンド用にユニークなバスインスタンスを確実に作成できます。
