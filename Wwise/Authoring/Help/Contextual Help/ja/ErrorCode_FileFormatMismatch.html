Source file is of different format than expected.

このエラーはストリーミングファイルや外部ソースが、関連する構造サウンドバンクまたは AK::SoundEngine::PostEvent() コールに記載されているのと同じコーデックを使用していない場合に発生します。

考えられる原因:

  • プロジェクトのConversion Settingsで、最近のコーデックの変更。

  • External Sourcesの場合、AK::SoundEngine::PostEvent()の呼び出しに対してAkExternalSourceInfo構造体に渡されたフォーマットで指定されたコーデックと実際のファイル形式が一致していません。

推奨される解決策:

  • すべてのサウンドバンクとWEMファイル（ストリーミングファイル）が生成され、対象コンソールに正しく展開されたことを確認します。

  • External Sourceを使う場合は、WSOURCESファイルの、指定されたConversion Settingsで使われるコーデックを確認します。
