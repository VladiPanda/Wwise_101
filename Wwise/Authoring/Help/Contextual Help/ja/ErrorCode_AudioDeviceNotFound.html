AkOutputDeviceID not found as an active device in call to function ...

このエラーは提供されたAkOutputDeviceIDに一致するサウンドエンジンが、使用中のデバイスがWwiseで見つからない場合に発生します。

推奨される解決策:

  • ゲームにデバッガを接続した状態で、同じシナリオをデバッグ版で再試行します。

  • 置き換え用に提供された AkOutputDeviceID が AK::SoundEngine::Init の際に使用した AkOutputDeviceID と一致することを確認します。提供された AkInitSettings::settingsMainOutput の audioDeviceShareset が 0 の場合、初期の AkOutputDeviceID は "System" という名前の ShareSet の ID と、提供された AkInitSettings::settingsMainOutput の idDevice のミックスであることにご注意ください。

  • 置き換え用に提供された AkOutputDeviceID が、以前 AK::SoundEngine::AddOutput() または AK::SoundEngine::ReplaceOutput() で返された AkOutputDeviceID と一致することを確認します。

  • この AkOutputDeviceID で、 AK::SoundEngine::RemoveOutput() へのコールが行われなかったことを、確認します。
