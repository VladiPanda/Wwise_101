AddOutput() needs unique Listeners for multi-instance outputs using the same Audio Device type.

Game Controllerのオーディオやモーションのように、同じ種類のAudio Deviceタイプを複数の出力で使用する場合は、Audio Deviceタイプだけでサウンドのルーティングを完結することはできません。同じAudio Deviceタイプで作成した各インスタンスに対し、それぞれ固有のリスナーを指定する必要があります。各出力で異なるオーディオミックスを実現するために必要です。

推奨される解決策:

  • 同じタイプを使う出力に対して、出力別にリスナーとして使用する新しいGame Objectを登録します。
