Maximum number of hardware-accelerated voices reached. Voice will not play.

ハードウェアアクセラレーションのオーディオフォーマットを使う、同時再生中のボイスが、多すぎます。

考えられる原因:

  • あなたが使っているコンバージョン設定が、同じオーディオフォーマットをソフトウェアでデコードするほかのプラットフォームと、同じコンバージョン設定です。一部のオーディオフォーマットは、特定のプラットフォームで、自動的にハードウェアアクセラレーションを使うことがあります。

  • You are using a large number of multichannel, hardware-accelerated sounds. On some platforms, the number of channels counts against the limit, which means you can decode more mono/stereo sources concurrently than 5.1 or 7.1 sources.

  • バーチャルボイスのスレッショルド設定が低すぎます。

推奨される解決策:

  • 一部のサウンドにおいて、別のコンバージョン設定を使います。さまざまなプラットフォームやユースケースにおけるそれぞれの最適なコンバージョン設定を知るには、 Audio Formats を参照してください。

  • ハードウェアアクセラレーションのオーディオフォーマットを使用する同時サウンドの数を制限します。

  • ハードウェアアクセラレーションのサウンドに、モノソースやステレオソースを使います。

  • バーチャルボイスのスレッショルドを上げます。
