Reset Playlist action with a game object specific scope is ignored on a global container.

SequenceやRandom Containerのプレイリストは常にゲームオブジェクト固有のものです。このため、ScopeプロパティがGlobalに設定されているContainerにReset Playlistアクションを含むEventをポストしても効果はありません。

推奨される解決策:

  • EventからReset Playlistアクションを削除します。

  • Container ScopeプロパティをGame Objectに変更します。
