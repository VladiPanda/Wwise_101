Memory allocation failed.

メモリ不足のため、メモリアロケーションが失敗しました。Wwiseが使用できる最大メモリは初期化時に AkMemSettings::uMemAllocationSizeLimit の設定によってコード内で定義します。Advanced ProfilerのMemoryタブには200ミリ秒ごとに取得した現在のメモリ状態のスナップショットが表示されます。つまり該当する時間にエラーが報告されているにも関わらず、充分なメモリがあると表示される可能性があります。

推奨される解決策:

  • 使用しないサウンドバンクをアンロードしたり、サウンドバンクを小さいファイルに分割したりすることで、使用するメモリを減らします。これはメモリを最も消費する部分をターゲットにしています。

  • Game Objectを登録解除することで、メモリ使用量を減らします。

  • AkMemSettings::uMemAllocationSizeLimit を増やします。
