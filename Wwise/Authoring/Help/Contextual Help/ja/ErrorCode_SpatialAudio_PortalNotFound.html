AK::SpatialAudio::SetPortalObstructionAndOcclusion または AK::SpatialAudio::SetGameObjectToPortalObstruction に渡されたPortal IDは、事前に AK::SpatialAudio::SetPortal を使用してSpatial Audioに登録されていませんでした。

推奨される解決策:

  • API Calls を Profiler Settings （Alt-G）で有効にしてゲームが行っているAPIコールのプロファイリングを行います。シナリオを再テストして、このエラーが発生する前に関数に送られたパラメータを確認します。

  • エラー文字列にIDが記載されているPortalを、 AK::SpatialAudio::SetPortal を使用してSpatial Audioに設定します。
