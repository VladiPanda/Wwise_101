Event ID not found.

PostEvent をコールされましたが、サウンドエンジンは指定したEventを見つけることができませんでした。同じラインに、使用したイベントが報告されます。

考えられる原因:

  • サウンドバンクがロードされていないか、正しくロードされていません。

  • SoundBankが最新のものではありません。

  • イベント名に、入力ミスがあります。

  • イベントIDに、入力ミスがあります。

推奨される解決策:

  • イベント名やIDがゲームコードで正しく書かれているかを確認する。

  • このエラーより先に発生したSoundBankのロード時のエラーを確認し、フィックスします。

  • このイベントがSoundBankに入っていて、そのSoundBankがロードされていることを、Advanced ProfilerのSoundBanksタブで確認します。

  • SoundBankが最新状態にあることを確認する。

  • ビルドプロセスにおいて、SoundBankがゲームと正しくパッケージングされていることを確認します。

  • ターゲットのコンソールやモバイルに、SoundBankが正しくデプロイされていることを確認します。
