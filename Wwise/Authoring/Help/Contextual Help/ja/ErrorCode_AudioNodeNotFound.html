Audio node not found in call.

指定の関数のパラメータで参照されるオーディオノードが、その時点でロードされているサウンドバンクの中に見つかりませんでした。

考えられる原因:

  • このオーディオノードの入ったサウンドバンクがまだロードされていなかいか、このコールの前にアンロードされました。

  • このオーディオノードがプロジェクトのどのサウンドバンクにも入っていません。

  • 関数パラメータで渡されたIDに誤字がありました。

  • オーディオノードをプロジェクトに追加してからサウンドバンクを再生成していなかった、または再展開していませんでした。

推奨される解決策:

  • Wwise Profilerを接続し、このエラーの前にそのオブジェクトを含むサウンドバンクがロードされているかどうかを確認します。

  • Profiler Settings （Alt-G）で API Calls を有効にします。Wwise Profilerを接続し、このエラーをトリガーする関数に渡しているパラメータを検証します。または、ゲームコード内の指定された関数コールにブレークポイントを設定します。

  • テストコンソールでバンクを再生成して再展開します。
