No memory size provided in AkExternalSource structure, but data pointer was used. Fill uiMemorySize.

AK::SoundEngine::PostEventを、External Sourcesやメモリに事前にロードしたデータで使用する場合は、構造体 AkExternalSourceInfo に対して有効なpInMemoryとuiMemorySizeの両方を提供する必要があります。

推奨される解決策:

  • 事前にロードしたWEMファイル全体のバイト数をuiMemorySizeに入れます。

  • szFileにファイル名を指定して、ディスクからストリーミングするバージョンを代わりに使用します。
