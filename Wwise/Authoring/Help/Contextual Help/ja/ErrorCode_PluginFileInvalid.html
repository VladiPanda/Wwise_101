Not a plug-in dynamic library.

A valid dynamic library was loaded, but it did not export the expected symbols for a Wwise Sound Engine plug-in.

考えられる原因:

  • プラグインコードに、 AK_IMPLEMEMENT_PLUGIN_FACTORY マクロが使われていません。

推奨される解決策:

  • プラグインに、 AK_IMPLEMENT_PLUGIN_FACTORY マクロを利用するコンパイルユニットを追加します。準拠したWwise Sound Engine Plug-inの書き方について、詳しくはWwise SDKドキュメンテーションの、 Sound Engineプラグインの概要 を参照してください。
