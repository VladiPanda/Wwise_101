Not enough memory in I/O pool to start stream.

I/Oプールのメモリが不足して、、ストリーミングファイルのヘッダーを読み込めません。この音は、終了されます。

推奨される解決策:

  • I/Oプールのメモリを増やします。初期化シーケンスで、Streaming Managerを初期化する前に、 AkDeviceSettings::uIOMemorySizeを設定します。

  • Wwiseプロファイラを接続してStreamingタブを確認する。ストリーミングボイス数が異常に多くないかを検証する。

  • バーチャルボイスやリミットを利用して、同時再生されるストリーミング音を減らします。

推奨される解決策:

  • 低ボリュームのオブジェクトの管理

  • オブジェクトの再生インスタンスの制限

  • Audiokinetic ストリームマネージャ初期化設定
