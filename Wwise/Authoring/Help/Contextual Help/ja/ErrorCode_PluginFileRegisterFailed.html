Could not register plugin.

The specified Dynamic Library for a plug-in was found on disk but there was an error while loading the library. ロード時に存在しないプラグインに関する、外部依存の典型的な症状です。

推奨される解決策:

  • Audiokineticが開発していないサードパーティプラグインの場合は、そのプラグインのドキュメントを読んで前提条件を確認します。

  • プラグインベンダーにサポート依頼をします。

  • 社内で開発されたプラグインの場合は、近くに座っている仲の良いプログラマーにお願いしてみましょう。
