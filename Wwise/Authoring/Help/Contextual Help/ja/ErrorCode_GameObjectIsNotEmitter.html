The game object for an event was not registered as an emitter. Make sure the game object is registered as an emitter before using it to post an event.

このエラーはInsufficient Memoryエラーなど、以前のエラーの後続として発生することがあります。Game Objectは存在しますが、その初期化が不完全であったことを意味します。

推奨される解決策:

  • Capture Logを調べて同じGame Objectに関連するほかのエラーを確認し、先にそれらのエラーを修正します。

  • エラーが続く場合は、Audiokineticサポートにご連絡ください。
