AK::SpatialAudio::SetGeometry - More than two triangles are connected to the same edge.

AK::SpatialAudio::SetGeometryで提供された三角形の1つが、正しく構築されていません。APIの要件により、1辺を共有できる三角形は2つまでです。

推奨される解決策:

  • API Calls を Profiler Settings （Alt-G）で有効にしてゲームが行っているAPIコールのプロファイリングを行います。シナリオを再テストして、このエラーが発生する前に関数に送られたパラメータを確認します。

    誤って同じ1つの辺を共有してしまった3つの三角形のTriangles配列のインデックスと、その辺を形成するVertices配列のインデックスが、エラー文字列に示されます。
