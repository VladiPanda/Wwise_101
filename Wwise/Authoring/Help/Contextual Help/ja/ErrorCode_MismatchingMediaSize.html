Mismatching media size error for file ... Possibly caused by mismatching sound bank or WEM resources.

このエラーは Zero Latency オプションを使用するサウンドにのみ発生します。WEMファイルのサイズがそれを参照するサウンドバンクの生成時のサイズと同一でないことをWwiseが検知したことを示します。これは通常サウンドバンクやWEMファイルの生成、パッケージング、または展開に問題があることを指摘しています。これは通常ゲームエンジンが対応するプロセスです。

考えられる原因:

  • ゲーム内のサウンドバンクファイルやWEMファイルのパッケージングまたは展開のエラー。ファイルの1つでもコピーに失敗すると、この問題が起きます。

  • 音が2つのサウンドバンクで複製されるため、Zero Latencyの部分も複製されます。1つのサウンドバンクだけが再生成されると、この問題が起きる可能性があります。

  • 古いサウンドバンクがゲームにロードされました。

  • このストリームを読み込む際にI/Oフックで問題が発生。カスタムI/Oフックの使用に関係することが、よくあります。

  • 内部バグ。

推奨される解決策:

  • サウンドバンクを再生成します。

  • オーディオファイルのキャッシュをクリアして、WEMファイルを再変換します。ファイルキャッシュをクリアにするには、Fileメニューで選択するか、.cacheフォルダの該当するWEMファイルをマニュアル操作で削除します。

  • テストコンソール上のすべての.bnkファイルと.wemファイルを削除します。

  • Conversion Settingsで別のコーデックを使用します。

  • この問題が続く場合は、Audiokineticサポートにご連絡ください。
