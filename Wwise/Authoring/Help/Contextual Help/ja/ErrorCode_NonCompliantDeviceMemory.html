Non-compliant device memory detected. Device memory is required for hardware acceleration.

サウンドエンジンが使用するMemory Managerが非対応のデバイスメモリを割り当てていました。デバイスメモリはプラットフォーム固有の仮想メモリセグメントで特別な属性を持ちます。これらの属性は特定のハードウェアアクセラレーションのデコード操作に必要です。これは通常ハードウェアオーディオデコーダを使用する場合に起こります。

考えられる原因:

  • ゲームは AK::MemoryMgr のデフォルト実装を使用していますが、デバイスメモリ割り当てコールバックの初期化設定が正しくありません。

  • ゲームは AK::MemoryMgr のカスタム実装を使用していますが、そのデバイスメモリ割り当て関数が誤って実装されています。

推奨される解決策:

  • SDKドキュメントのデフォルトMemory Managerを適切にオーバーライドする方法 でMemory Managerの要件を確認し、ゲームが完全に準拠していることを検証します。
