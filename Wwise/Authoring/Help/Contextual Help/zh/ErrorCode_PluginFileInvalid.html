Not a plug-in dynamic library.

A valid dynamic library was loaded, but it did not export the expected symbols for a Wwise Sound Engine plug-in.

可能的原因：

  • 插件代码未使用 AK_IMPLEMEMENT_PLUGIN_FACTORY 宏。

推荐的解决步骤：

  • 将编译单元添加到使用 AK_IMPLEMENT_PLUGIN_FACTORY 宏的插件。有关如何编写相符 Wwise 声音引擎插件的更多详细信息，请参阅 Wwise SDK 文档中的声音引擎插件概述章节。
