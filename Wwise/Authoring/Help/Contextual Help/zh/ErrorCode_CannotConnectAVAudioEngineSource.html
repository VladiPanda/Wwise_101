Failed to connect System Audio Object to Apple Spatial Audio. Some sounds will not be audible.

The Audio Device has received input for a new System Audio Object, but is unable to connect it to the audio output graph as a Apple Spatial Audio Point Source. The Audio Device will keep attempting to connect the object as long as it is playing, so the beginning part of the sound might be silent, or it might not be audible at all.

This only occurs on macOS, iOS, or tvOS when using the 3D Audio pipeline, which is implemented using Apple Spatial Audio.

可能的原因：

  • There are too many point sources for Apple Spatial Audio to handle.

推荐的解决步骤：

  • Reduce the number of allocated Spatial Audio point sources in AkPlatformInitSettings::uNumSpatialAudioPointSources.
