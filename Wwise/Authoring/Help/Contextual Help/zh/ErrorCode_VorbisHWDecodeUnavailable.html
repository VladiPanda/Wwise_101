Vorbis Hardware Acceleration library not found. Source will be decoded in software.

“未找到 Vorbis Hardware Acceleration 库。将在软件中对音频源进行解码。”在初始化声音引擎时启用了 Vorbis Hardware Acceleration（Vorbis 硬件加速）选项，但没有加载所需的 PRX 库。

在加载该库之前，所有 Vorbis 音频源都会显示此错误，然后在 CPU 上进行解码。

可能的原因：

  • 游戏未调用 AK::LoadVorbisHwAcceleratorLibrary API 函数。

  • 没有随游戏一并正确打包 PRX 库。

推荐的解决步骤：

  • 在播放 Vorbis 音频源之前，确保游戏通过 AK::LoadVorbisHwAcceleratorLibrary API 调用正常加载 Vorbis Hardware Acceleration 库。该函数只需调用一次，并且可在声音引擎初始化前调用。确保函数调用返回成功。
