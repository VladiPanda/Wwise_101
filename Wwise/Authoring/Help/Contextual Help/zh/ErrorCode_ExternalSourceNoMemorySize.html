No memory size provided in AkExternalSource structure, but data pointer was used. Fill uiMemorySize.

When using AK::SoundEngine::PostEvent with External Sources and preloaded data in memory, you must provide both a valid pInMemory and uiMemorySize for the structure AkExternalSourceInfo.

推荐的解决步骤：

  • 使用预加载 WEM 文件的全部字节数填写 uiMemorySize。

  • 改为在 szFile 中指定文件名称，以便使用从磁盘流播放的版本。
