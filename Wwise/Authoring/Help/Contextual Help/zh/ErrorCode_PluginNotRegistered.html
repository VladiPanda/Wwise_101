Plug-in not found.

“未找到插件”。此错误表示游戏所加载的 SoundBank（音频包）使用了未知的源、效果器、混音器或 sink 插件。因此，无法找到可执行此插件的代码。

可能的原因：

  • The game code doesn't static link with the plug-in library. 所有 Unreal 游戏和大部分自研引擎都可能出现此问题。

  • The game cannot find the plug-in dynamic library (DLL or SO). 所有 Unity 游戏和部分自研引擎都可能出现此问题。

  • The game cannot load the plug-in dynamic library (DLL or SO) because of a missing dependency. 查看相关插件文档中的要求。

  • SoundBank 没有更新。

  • SoundBank 已损坏。

  • Init.bnk 没有加载调用 AK::SoundEngine::GetDeviceList 的插件，或者没有通过 AK::SoundEngine::RegisterPlugin 显式调用该插件。

推荐的解决步骤：

  • 若游戏与 Wwise 库静态链接，请确认游戏的 #include "ThePluginFactory.h" 和链接选项。详见： 集成详情 – 插件

  • 若游戏使用动态链接（比如 Unity），请重新生成 SoundBank 并构建游戏。

  • 重新生成 SoundBank，并重新打包到游戏中（包括 Init.bnk）。

  • 对于 Sink 插件，确保此插件的共享集关联有至少一条主总线。
