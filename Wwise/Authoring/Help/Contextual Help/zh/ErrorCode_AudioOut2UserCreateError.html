sceAudioOut2UserCreate Failed.

There was a problem with the userId required to initialize an output device for a specific user, such as a personal or controller speaker output device. An error code of -2144960496, or SCE_AUDIO_OUT2_ERROR_INVALID_USER (0x80268010) is typically returned. The userId specified for personal and controller speaker outputs must appear in the list of logged-in users provided by sceUserServiceGetLoginUserIdList. The userId itself is provided by AkOutputSettings::idDevice. Note that when initializing a Motion output device, the value for AkOutputSettings::idDevice must be a handle for an scePad, which must map to a currently logged-in user, and not a userId.
