Plug-in version doesn't match sound engine version. Ensure the plug-in is compatible with this version of Wwise.

이 오류는 컴파일링 환경이 잘못되었거나 Unity의 경우 통합이 잘못되었을 때 일어납니다. 이 오류는 항상 최근 Wwise 업데이트로 인해 일어납니다. 모든 플러그인 유형 (Effect, Source 등)은 게임 코드와 연결되어야 하는 게임 측 라이브러리를 가집니다. 이 오류는 게임 측 플러그인 코드가 나머지 사운드 엔진 라이브러리와 대응하지 않을 경우에 일어납니다. 게임 이진법은 여전히 제대로 연결할 수 있지만 런타임 때 작동할 수 없습니다.

유력한 원인:

  • 파일 권한으로 인해 오래된 라이브러리가 업데이트되지 않았습니다.

  • Unity에서 파일 권한으로 인해 Plugin 폴더에 오래된 DLL/SO/BUNDLE/PRX가 있습니다.

  • Unity에서 콘솔/모바일에 오래된 DLL/SO/BUNDLE/PRX가 있어서 새로운 파일로 교체하지 못합니다.

권장 해결 단계:

  • Wwise/SDK/[Platform]/lib 폴더를 확인하고 라이브러리가 모두 동일한 타임스탬프에 있는지 확인하세요.

  • Unity에서 Wwise/Plugins/DSP 폴더를 확인하고 라이브러리가 모두 동일한 타임스탬프에 있는지 확인하세요.

  • 깨끗한 게임 빌드를 제작하세요.

  • 대상 콘솔/모바일의 배포 폴더를 비운 후 게임을 재배포하세요.
