Not a valid Switch value ... in group ...

Switch Container가 유발되었지만 어떤 사운드도 재생할 수 없었음을 말합니다.

유력한 원인:

  • 프로젝트에서 Assigned Objects 창의 Switch Container에 있는 Switch Values와 관련된 사운드가 없습니다.

  • 프로젝트에서 Switch Container의 속성에 'Default Switch/State'가 정의되지 않았으며 게임의 Game Object에서 Switch Value가 설정되지 않았습니다.

  • 게임에서 게임 오브젝트에 Switch Value가 설정되었지만 'Assigned Object' 섹션에 이와 연결된 Assigned Object가 없습니다.
