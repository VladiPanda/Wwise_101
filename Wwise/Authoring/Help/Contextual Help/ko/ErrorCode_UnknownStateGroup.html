Unknown State Group referenced.

One or more items contained in a Sounbank refer to an unknown State Group. Known State Groups are contained in the Init.bnk. This issue is most likely caused by an out-of-date Init.bnk.

Rather than fail the loading of the Soundbank, the State Group is created internally with a default transition time of 0s. The State Group may be referenced by other loaded objects as well as API function calls.

권장 해결 단계:

  • Regenerate Soundbanks.
