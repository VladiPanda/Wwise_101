Non-compliant device memory detected. Device memory is required for hardware acceleration.

사운드 엔진에서 사용하는 Memory Manager가 비호환 장치 메모리를 할당했습니다. 장치 메모리는 특별한 속성을 가진 플랫폼별 가상 메모리 세그먼트입니다. 이러한 속성은 특정 하드웨어 가속 디코딩 작업에 필요합니다. 이 오류는 일반적으로 하드웨어 오디오 디코더를 사용하는 경우에 일어납니다.

유력한 원인:

  • 게임이 AK::MemoryMgr의 기본 구현을 사용하지만 장치 메모리 할당 콜백에 대해 잘못된 초기화 설정을 제공했습니다.

  • 게임이 장치 메모리 할당 기능을 잘못 구현하는 커스텀 구현 AK::MemoryMgr를 사용합니다.

권장 해결 단계:

  • SDK 문서의 기본 Memory Manager를 올바르게 오버라이딩하기에서 Memory Manager의 요구 사항을 검토하고 게임이 완전히 호환되는지 확인하세요.
