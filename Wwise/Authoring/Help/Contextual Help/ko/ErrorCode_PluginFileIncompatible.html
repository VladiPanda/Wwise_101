Incompatible plug-in dynamic library file.

파일 형식이 플랫폼의 표준 동적 라이브러리 형식과 대응하지 않아서 플러그인 동적 라이브러리를 로드하는 데에 실패했습니다.

유력한 원인:

  • 게임 파일 패키징 도중 파일이 손상되었습니다.

  • 패키징된 라이브러리가 다른 플랫폼용으로 컴파일되었습니다.

  • 라이브러리가 플랫폼의 펌웨어가 지원하는 버전보다 오래된 SDK 버전을 사용하여 컴파일되었습니다.

권장 해결 단계:

  • 프로젝트의 패키징 단계를 검토하여 대상 플랫폼에 알맞은 라이브러리 파일이 복사되었는지 확인하세요.

  • 파일 해시 장치를 사용하여 파일 복사 작업을 검토해서 패키징 도중 파일 손상 오류가 일어나는지 확인하세요.

  • 플러그인을 컴파일링할 경우 대상 플랫폼용 최신 SDK를 사용하고 있는지 확인하세요.
