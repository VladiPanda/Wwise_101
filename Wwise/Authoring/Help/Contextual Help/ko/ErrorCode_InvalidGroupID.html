Invalid State Group ID.

AK::SoundEngine::SetState 호출에 사용된 State Group 이름이나 ID 숫자를 찾을 수 없었습니다.

유력한 원인:

  • Init SoundBank가 아직 로딩되지 않았거나 올바르게 로딩되지 않았습니다.

  • Init SoundBank가 최신 버전이 아니며 최신 변경 사항을 담고 있지 않습니다.

  • 코드에서 State Group의 이름이나 ID에 오타가 있습니다.

권장 해결 단계:

  • Wwise Profiler의 SoundBanks 탭을 확인하여 Init SoundBank가 로딩되었는지 확인하세요.

  • SoundBank를 재생성하고 대상 콘솔에 재배포하세요.

  • State Group의 이름을 확인하세요.
