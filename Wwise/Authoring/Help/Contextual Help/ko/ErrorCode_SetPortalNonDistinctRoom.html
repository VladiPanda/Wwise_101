AK::SpatialAudio::SetPortal: Portal must have a front room which is distinct from its back room.

Portal은 반드시 두 개의 다른 Room으로 연결되어야 합니다. AkPortalParams::FrontRoom과 AkPortalParams::BackRoom을 통해 AK::SpatialAudio::SetPortal에 전달된 Room ID가 동일합니다.

권장 해결 단계:

  • Profiler Settings(Alt-G)에서 API Calls을 활성화하여 게임이 수행하는 API 호출을 프로파일링하세요. 상황을 다시 시험하여 이 오류 이전에 해당 기능에 제공된 매개 변수 값을 확인하세요.

  • AkPortalParams::FrontRoom으로 지정한 Room과 AK::SpatialAudio::SetPortal에 전달된 AkPortalParams::BackRoom은 반드시 서로 달라야 합니다.
