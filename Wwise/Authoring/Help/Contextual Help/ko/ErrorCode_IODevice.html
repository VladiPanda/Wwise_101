Unknown IO device error. .. returned .. on file ...

이 오류는 I/O 장치에서 Read나 Write 작업 중 오류가 발생함을 말합니다. The actual error is device-specific and reported in the message. 이 오류는 게임이 자체적으로 I/O 연결을 구현할 경우에 일어날 수 있습니다.

권장 해결 단계:

  • I/O 연결 코드에 디버거를 첨부하세요.

  • If you do not have a custom I/O hook, contact Audiokinetic Support.
