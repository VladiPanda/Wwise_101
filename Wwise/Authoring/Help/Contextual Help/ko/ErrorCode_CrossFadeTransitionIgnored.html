Cross-Fade transition ignored: Estimated length not available. Make sure sounds with Cross-Fade transitions virtual mode are set to "Play from elapsed time" and that only sounds with finite duration are used.

매개 변수의 크로스페이드를 계산하려면 사운드의 실제 길이가 필요합니다. 무제한 반복 사운드는 끝나지 않기 때문에 크로스페이드될 수 없습니다. 사운드의 Virtual Mode가 Play From Beginning과 Resume 옵션을 사용할 경우 사운드의 실제 길이에 영향을 줄 수 있기 때문에 미디어의 실제 길이보다 사운드가 더 오래 지속될 수 있습니다.

이 경우 From 사운드나 To 사운드 모두에서 크로스페이드가 일어나지 않게 됩니다.

권장 해결 단계:

  • Validate the Virtual Voice Behavior in the Property Editor used for the sound associated with this error. 'Play from Elapsed' 혹은 'Continue To Play'만 사용하세요.

  • 사운드가 무제한 반복 재생되지는 않은지 확인하세요.
