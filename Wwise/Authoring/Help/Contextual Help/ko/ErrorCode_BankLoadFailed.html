Bank Load Failed.

SoundBank 불러오기에 실패했다는 일반 오류 메시지입니다. 해당 뱅크에 포함된 콘텐츠를 재생할 준비가 되지 않았습니다.

유력한 원인:

  • SoundBank 파일을 찾지 못했을 때 (이 경우 File Not Found 오류가 먼저 뜨게 됩니다).

  • SoundBank가 다른 버전의 Wwise로 생성되었을 때 (이 경우 Wrong Bank Version 오류가 먼저 뜨게 됩니다). 이 오류는 해당 SoundBank와 SDK 버전을 표시합니다.

  • SoundBank 파일이 손상되었습니다.

  • SoundBank를 불러오는 중 I/O 오류가 일어났습니다.

권장 해결 단계:

  • Wwise가 제공하는 기본 I/O 코드를 사용할 경우 SetBasePath(혹은 AddBasePath)로의 호출이 올바른 폴더로 향하는지 확인하세요.

  • SoundBank가 올바른 Wwise 버전으로 제대로 생성되었는지 확인하세요.

  • Unity나 Unreal을 사용할 경우 게임 엔진 안에 있는 Wwise Integration이 Wwise 저작 도구와 동일한 버전으로 업데이트되었는지 확인하세요.

  • SoundBank가 올바르게 패키징되어 대상으로 배포되었는지 확인하세요.

  • 맞춤 I/O 연결을 사용했을 경우 맞춤 I/O 연결 코드를 디버그하세요.
