AddOutput() needs unique Listeners for multi-instance outputs using the same Audio Device type.

Game Controller 오디오나 모션의 경우와 같이 여러 출력에 동일한 Audio Device 유형을 사용하는 경우 Audio Device 유형만으로는 사운드 라우팅을 해결할 수 없습니다. 반드시 동일한 Audio Device를 사용하여 생성한 각 인스턴스에 고유한 리스너를 지정해야 합니다. 이 작업은 각 출력마다 서로 다른 오디오 믹스를 만들어내기 위해 꼭 필요한 작업입니다.

  • 동일한 유형을 사용하여 각 출력에 대한 리스너로 사용할 새 Game Object를 등록하세요.
