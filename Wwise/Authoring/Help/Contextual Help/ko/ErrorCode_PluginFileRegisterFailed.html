Could not register plugin.

The specified Dynamic Library for a plug-in was found on disk but there was an error while loading the library. 이 오류는 플러그인의 외부 종속성이 불러오기 작업 시에 존재하지 않을 경우 흔히 일어납니다.

권장 해결 단계:

  • 서드 파티 플러그인인 경우 (Audiokinetic이 제작하지 않았을 경우) 해당 플러그인에 대한 설명서를 읽고 전제 조건을 검토하세요.

  • 플러그인 판매자에게 지원을 요청하세요.

  • 사용자의 회사에서 내부적으로 개발한 플러그인일 경우 회사 안의 프로그래머에게 도움을 요청하세요.
