The playing sound is assigned the same Early Reflection Auxiliary Bus in the Authoring Tool that has been set via AK::SpatialAudio::SetImageSource. Use a unique bus to avoid image source conflicts.

Wwise Authoring에서 사운드에 할당된 Early Reflections Auxiliary Bus가 AK::SpatialAudio::SetImageSource로 전달된 버스와 충돌합니다.

유력한 원인:

  • Wwise Authoring에서 사운드에 할당된 Early Reflections Auxiliary Bus가 동일한 버스를 가지며 API 함수 AK::SpatialAudio::SetImageSource에 전달된 동일한 게임 오브젝트에서 재생됩니다. Reflect의 단일 인스턴스는 지오메트리 API와 SetImageSource API 모두에서 허음원(image source)데이터를 수신할 수 없습니다.

권장 해결 단계:

  • 지오메트릭 반사 API와 SetImageSource API를 함께 사용하는 경우 Wwise Authoring에서 동일한 게임 오브젝트에서 재생될 사운드에 정의된 Auxiliary Bus와 다른 고유한 Auxiliary Bus를 SetImageSource에 지정하세요. Reflect 플러그인이 포함된 원하는 Auxiliary Bus를 복사하여 충돌하는 사운드에 지정하면 됩니다. 이렇게 하면 해당 사운드에 고유한 버스 인스턴스가 생기게 됩니다.
