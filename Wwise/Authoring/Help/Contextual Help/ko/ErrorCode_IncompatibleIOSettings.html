I/O settings incompatible. 블록 크기는 1과 Granularity 및 Granularity의 전체 분수 사이여야 합니다.

AkDeviceSettings에 제공된 I/O 설정이 불완전하거나 유효하지 않습니다.

유력한 원인:

  • Block Size는 반드시 1과 Granularity 설정 사이의 값이어야 합니다. Granularity 설정은 반드시 Block Size의 짝수 배수여야 합니다.

  • uBufferSize가 플랫폼의 버퍼 정렬 (보통 16바이트)과 정렬되지 않습니다

권장 해결 단계:

  • AK::StreamMgr::Create를 호출할 때 사용된 AkDeviceSettings를 검토하세요.

  • Debug에서 게임을 컴파일하고 상황을 재실행해보세요. Audiokinetic 지원팀에 어서트를 모두 보고해주세요.

  • 게임이 자체 버전의 Stream Manager 혹은 I/O 연결을 구현할 경우 디버그를 부착해서 오류를 추적하세요.
