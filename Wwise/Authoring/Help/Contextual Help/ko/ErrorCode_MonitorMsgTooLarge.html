Wwise Profiler에서 사용하는 Monitor Queue가 Profiler로 보낼 메시지 중 하나를 포함할 수 없습니다. 메시지의 최대 크기는 AkInitSettings::uMonitorQueuePoolSize의 절반입니다. 이 오류는 특정 API가 매개 변수의 무제한 배열을 전달할 수 있으므로 API 호출을 프로파일링할 때 트리거될 수 있습니다. 이 오류의 유일한 부작용은 Wwise Profiler에서 메시지가 누락된다는 것입니다.

  • 매개 변수 배열의 수를 줄이기 위해 AK::SoundEngine::SetMultiplePositions 및 모든 Spatial Audio 지오메트리 함수에 대한 호출을 확인하세요. 이 메시지는 종종 API 호출의 형식이 잘못되었음을 의미합니다.
