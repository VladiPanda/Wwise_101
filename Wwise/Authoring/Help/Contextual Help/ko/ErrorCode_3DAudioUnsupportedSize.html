AkInitSettings::uNumSamplesPerFrame set to a value that is not 256, 512, or 1024. 3D Audio will be disabled.

System Audio Device 속성은 3D 오디오를 허용하도록 지정하지만 오디오 프레임 크기는 3D 오디오 하드웨어와 호환되지 않습니다. 256, 512 또는 1024 샘플의 오디오 프레임 크기가 필요합니다. 다른 값을 사용하면 이 오류가 발생합니다.

권장 해결 단계:

  • AK::SoundEngine::Init을 호출하기 전에 AkInitSettings::uNumSamplesPerFrame을 256, 512 또는 1024로 변경하세요.

  • 3D Audio를 사용하지 않으려면 Wwise 프로젝트의 System Audio Device 설정에서 Allow 3D Audio를 선택 해제하세요.
