Seeking after end of file Playback will stop.

이 오류는 AK::SoundEngine::SeekOnEvent를 사용할 때 SeekPercent 매개 변수가 100%보다 높거나 Position 매개 변수가 사운드 파일의 길이보다 더 길 경우 일어날 수 있습니다.

권장 해결 단계:

  • AK::SoundEngine::SeekOnEvent 함수의 매개 변수를 검토하세요. Profiler Settings에서 API Calls를 활성화한 경우 Capture Log에서 오류 근처에 잘못된 호출 매개 변수를 찾을 수 있습니다.

  • 사운드 파일의 길이를 확인하세요. 해당 오류 줄은 문제를 일으키는 사운드를 참조합니다. 'Wwise Object' 열을 더블 클릭해서 사운드에 접근하세요.
