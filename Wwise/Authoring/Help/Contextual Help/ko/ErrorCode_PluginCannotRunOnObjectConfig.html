Plug-in does not support Audio Objects bus configuration.

이 오류는 열거된 Effect 혹은 Object Processor 플러그인이 버스 구성이 Audio Objects인 버스에서 인스턴스화되었으며, 이것이 이 플러그인에서 유효한 작업 모드가 아님을 의미합니다. 해당 버스의 Property Editor에서 버스 구성이 명시적으로 Audio Objects로 설정되었거나 버스의 다른 플러그인에 의해 내재적으로 설정되었습니다.

권장 해결 단계:

  • 속성 페이지에서 구성을 알맞은 채널 구성(Audio Objects가 아닌 다른 구성)으로 명시적으로 설정해서 지원되는 구성으로 버스를 변경하거나, Effects 목록에서 앞에 있는 플러그인을 검토해 보세요. 그중 하나가 구성을 Audio Objects로 변경하고 있을 수도 있습니다.
