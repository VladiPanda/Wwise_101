Switch ... is bound to RTPC ... It can not be set directly.

이 오류는 Event나 함수 호출을 통해 Switch 값이 명시적으로 설정되지만 이 Switch 그룹이 RTPC에 의해 구동되는 경우에 발생합니다. 해당 Switch 값이 무시됩니다.

권장 해결 단계:

  • Event의 Set Switch Action 에서 해당 Switch Group을 사용하지 마세요.

  • SetSwitch 함수 호출에서 해당 Switch Group을 사용하지 마세요.

  • Deselect the Use Game Parameter option in the Switch Group definition in the Wwise Project.
