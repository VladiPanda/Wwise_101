Soundbanks have been generated with Convolution Reverb parameters that do not match Sound Engine runtime conditions. No wet path will be heard.

Pertains to AK Convolution Reverb. This error occurs if the Impulse Response file has a different sample rate than the Sound Engine.

권장 해결 단계:

  • Convolution Reverb Impulse Responses are generated for 48KHz playback in the Sound Engine by default. It will be necessary to force the Sounde Engine sample rate, at initialization time, by calling AK::SouneEngine::Init.
