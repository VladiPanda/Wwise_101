Audio Device not found in Init bank: ...

API 호출에서 찾을 수 없는 Audio Device를 참조했습니다.

유력한 원인:

  • Init 뱅크가 아직 로드되지 않았습니다.

  • 함수 매개 변수에 전달된 이름에 오타가 있습니다.

  • 프로젝트에 Audio Device를 추가한 후 Init 뱅크가 재생성되거나 재배포되지 않았습니다.

권장 해결 단계:

  • Wwise Profiler를 연결하여 이 오류가 발생하기 전에 Init bank가 로드되었는지 확인하세요.

  • Profiler Settings(Alt-G)에서 API Calls를 활성화하세요. Wwise Profiler를 연결하고 이 오류를 일으키는 함수에 전달된 매개 변수를 확인하세요.

  • 테스트 콘솔에서 뱅크를 재생성하고 재배포하세요.
