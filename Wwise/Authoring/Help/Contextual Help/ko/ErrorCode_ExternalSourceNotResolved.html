External source missing from PostEvent call.

AK::SoundEngine::PostEvent로의 호출이 하나 이상의 External Source 파일 교체가 필요한 Event를 사용하여 실행되었습니다. API 호출에 필요한 모든 음원이 제공되지 않았습니다. 한 Event에 여러 External Source가 있을 수 있으며 런타임 때 모든 음원을 코드로 제공해야 합니다. External Source ID도 반드시 일치해야 합니다.

권장 해결 단계:

  • Wwise 프로젝트에서 해당 Event의 대상이 External Source가 얼마나 필요한지 확인하세요. 문자열이나 식별 번호를 메모해두세요. 이 ID를 반드시 PostEvent에 제공해야 합니다. PostEvent 호출 시 배열에 동일한 수의 External Source가 있는지, 해당 ID가 Wwise 프로젝트의 ID와 일치하는지 확인하세요.
