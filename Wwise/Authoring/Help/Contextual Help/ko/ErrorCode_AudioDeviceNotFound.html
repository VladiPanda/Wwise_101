AkOutputDeviceID not found as an active device in call to function ...

이 오류는 제공된 AkOutputDeviceID와 일치하는 사운드 엔진에서 Wwise가 사용 중인 기기를 찾을 수 없을 때 발생합니다.

권장 해결 단계:

  • Debug 버전을 사용하여 게임에 디버거를 첨부한 채 동일한 상황을 재시도해보세요.

  • 교체를 위해 제공되는 AkOutputDeviceID가 AK::SoundEngine::Init 도중 사용되는 AkOutputDeviceID와 제대로 일치하는지 확인하세요. 제공된 AkInitSettings::settingsMainOutput에서 audioDeviceShareset가 0일 경우 초기 AkOutputDeviceID는 'System'이라는 ShareSet의 ID와 제공된 AkInitSettings::settingsMainOutput의 idDevice를 합친 것으로 설정됩니다.

  • 교체된 AkOutputDeviceID가 이전에 AK::SoundEngine::AddOutput()이나 AK::SoundEngine::ReplaceOutput()에서 반환된 AkOutputDeviceID와 일치하는지 확인하세요.

  • 문제의 AkOutputDeviceID가 포함된 AK::SoundEngine::RemoveOutput() 호출이 실행되지 않았는지 확인하세요.
