AK::SpatialAudio::SetGeometry - Triangle number ... is skipped because it is invalid; two or more of its vertices are at the same position.

AK::SpatialAudio::SetGeometry에 제공된 삼각형이 제대로 정의되지 않았습니다.

유력한 원인:

  • 이 문제는 같은 삼각형에서 좌표가 같은 꼭짓점을 사용하여 선이나 점만 정의하게 되면서 발생합니다.

권장 해결 단계:

  • Profiler Settings(Alt-G)에서 API Calls을 활성화하여 게임이 수행하는 API 호출을 프로파일링하세요. 상황을 다시 시험하여 이 오류 이전에 해당 기능에 제공된 매개 변수 값을 확인하세요.

    오류 메시지의 숫자는 Triangle 배열의 삼각형 색인과 Vertex 배열의 해당 꼭지점 색인을 나타냅니다.
