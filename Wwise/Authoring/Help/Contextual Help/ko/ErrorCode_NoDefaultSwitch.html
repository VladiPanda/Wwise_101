No default Switch value selected in group ...

Switch Container가 Switch가 설정되지 않은 Game Object에서 재생되도록 설정되었습니다. 따라서 기본 Switch로 되돌아가기를 시도했지만 Wwise 프로젝트에서 기본 Switch 값이 선택되지 않았습니다. Game Object에 항상 Switch 값이 설정되어 있는 한에서는 기본 Switch 값을 할당되지 않은 채로 둘 수 있습니다.

유력한 원인:

  • Switch Container에 기본 Switch 값이 설정되어 있지 않습니다.

  • Event가 게시되기 전에 Game Object에 Switch 값이 설정되지 않았습니다.

  • Switch Container에 기본값이 선택되어 있지만 SoundBank가 최신 상태가 아닙니다.

  • Switch Container를 완전히 생성하는 데 메모리가 충분하지 않았습니다.

권장 해결 단계:

  • Switch Container에 기본 Switch 값이 설정되어 있는지 확인하세요.

  • 이 Switch Container가 재생 중인 Game Object에 Switch 값이 설정되어 있는지 확인하세요.

  • 음악 SoundBank를 재생성하고 재배포하세요.

  • Capture Log에 'Memory allocation failed' 오류가 표시되는지 확인하세요. 그렇다면 여기에서 해당 오류를 수정하는 방법에 대해 읽어보세요:“Memory allocation failed.”
