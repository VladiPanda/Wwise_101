Type mismatch while loading bank. Object ... is a ... in the currently loading bank. It was a ... in bank ...

이 오류는 Wwise에서 사용되는 고유 식별자가 충돌함을 나타냅니다. 두 오브젝트가 두 개의 다른 SoundBank에서 동일한 식별자를 공유합니다.

유력한 원인

  • 일반적으로 콘솔 테스트 폴더에는 오래되고 사용되지 않은 SoundBank가 있습니다.

  • 실제로 식별자가 충돌될 수 있지만 극히 드뭅니다.

권장 해결 단계:

  • 테스트 콘솔에서 모든 .bnk 파일을 삭제하세요.

  • Wwise Project에서 보고된 오브젝트의 이름을 변경하고 SoundBank를 재생성하세요.

  • 내부 버그일 수도 있으므로 Audiokinetic 지원팀에 연락해 주세요.
