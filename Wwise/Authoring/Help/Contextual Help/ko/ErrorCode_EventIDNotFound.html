Event ID not found.

PostEvent가 호출되었지만 사운드 엔진이 지정된 Event를 찾을 수 없습니다. 사용된 Event가 같은 줄에 보고될 것입니다.

유력한 원인:

  • SoundBank가 로딩되지 않았거나 제대로 로딩되지 않았습니다.

  • SoundBank가 최신 버전이 아닙니다.

  • Event 이름에 오타가 있습니다.

  • Event ID에 오타가 있습니다.

권장 해결 단계:

  • 이벤트 이름 또는 ID를 게임 코드에 올바르게 적었는지 확인해 주세요.

  • 이 오류 전에 표시된 SoundBank 불러오기 오류를 확인하고 해결하세요.

  • Advanced Profiler의 SoundBank 탭에서 SoundBank에 해당 Event가 있는지, 그리고 SoundBank가 로딩되었는지 확인하세요.

  • SoundBank가 모두 최신 버전인지 확인해 주세요.

  • 생성 과정 중 SoundBank가 게임에 올바르게 패키징되었는지 확인하세요.

  • SoundBank가 대상 콘솔/모바일에 올바르게 배포되었는지 확인하세요.
