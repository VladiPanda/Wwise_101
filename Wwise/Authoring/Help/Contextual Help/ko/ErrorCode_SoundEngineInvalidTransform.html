Invalid transform in function ...

지정된 함수에 잘못된 방향이 전달되었습니다.

유력한 원인:

  • AkTransform의 방향 벡터인 Front와 Top이 서로 직교하지 않습니다.

  • AkTransform의 방향 벡터인 Front와 Top 중 하나 혹은 모두의 표준이 (길이) 1이 아닙니다.

권장 해결 단계:

  • Profiler Settings(Alt-G)에서 API Calls을 활성화하여 게임이 수행하는 API 호출을 프로파일링하세요. 상황을 다시 시험하여 이 오류 이전에 해당 기능에 제공된 매개 변수 값을 확인하세요.
