Effect ShareSet not found in function ...

지정된 함수의 매개 변수가 참조하는 Effect ShareSet를 당시 로드된 SoundBank에서 찾을 수 없었습니다.

유력한 원인:

  • 이 Effect ShareSet를 포함하는 SoundBank가 아직 로드되지 않았거나 호출 전에 언로드되었습니다.

  • Effect ShareSet는 프로젝트의 SoundBank의 일부가 아닙니다.

  • 함수 매개 변수에 전달된 ID에 오타가 있습니다.

  • Effect ShareSet를 프로젝트에 추가한 후 SoundBank가 재생성되거나 재배포되지 않았습니다.

참고

Bus가 사용된 Audio Device ShareSet로 이어지는 Effect ShareSet의 경우 Init SoundBank에 패키징됩니다. 그렇지 않으면 Effect ShareSet를 User-Defined SoundBank에 수동으로 추가하고 해당 SoundBank를 게임에서 로드해야 합니다.

권장 해결 단계:

  • Wwise Profiler를 연결하여 이 오류가 발생하기 전에 ShareSet가 포함된 SoundBank가 로드되었는지 확인하세요.

  • Profiler Settings에서 API Calls를 활성화하세요. Wwise Profiler를 연결하고 이 오류를 일으키는 함수에 전달된 매개 변수를 확인하세요. 또는 지정된 함수 호출에 대해 게임 코드에 중단점(breakpoint)을 넣으세요.

  • 테스트 콘솔에서 뱅크를 재생성하고 재배포하세요.
