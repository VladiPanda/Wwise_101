The game object for an event was not registered as an emitter. Make sure the game object is registered as an emitter before using it to post an event.

이 오류는 Insufficient Memory 오류와 같이 이전에 생겼던 오류에 대한 후속 오류로 발생할 수 있습니다. Game Object가 존재하지만 초기화가 완료되지 않았음을 의미합니다.

권장 해결 단계:

  • 동일한 Game Object와 관련된 다른 오류가 있는지 Capture Log를 검사하고 해당 오류를 먼저 수정하세요.

  • 이 문제가 계속 생길 경우 Audiokinetic 지원팀에 연락해 주세요.
