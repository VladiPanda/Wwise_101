Audio node not found in call.

지정된 함수의 매개 변수가 참조하는 오디오 노드가 그 당시 로드된 SoundBank에서 발견되지 않았습니다.

유력한 원인:

  • 이 오디오 노드를 포함하는 SoundBank가 아직 로드되지 않았거나 호출 전에 언로드되었습니다.

  • 이 오디오 노드가 프로젝트의 SoundBank에 속하지 않습니다.

  • 함수 매개 변수에 전달된 ID에 오타가 있습니다.

  • 프로젝트에 오디오 노드를 추가한 후 SoundBank가 재생성되거나 재배포되지 않았습니다.

권장 해결 단계:

  • Wwise Profiler를 연결하여 이 오류가 발생하기 전에 해당 오브젝트가 포함된 SoundBank가 로드되었는지 확인하세요.

  • Profiler Settings(Alt-G)에서 API Calls를 활성화하세요. Wwise Profiler를 연결하고 이 오류를 일으키는 함수에 전달된 매개 변수를 확인하세요. 또는 지정된 함수 호출에 대해 게임 코드에 중단점(breakpoint)을 넣으세요.

  • 테스트 콘솔에서 뱅크를 재생성하고 재배포하세요.
