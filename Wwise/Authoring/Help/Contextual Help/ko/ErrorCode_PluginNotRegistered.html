Plug-in not found.

이 오류는 게임이 알 수 없는 음원, 효과, 믹서, 혹은 싱크 플러그인을 사용하는 SoundBank를 불러왔음을 의미합니다. 게임이 이 플러그인을 실행할 코드를 찾지 못했습니다.

유력한 원인:

  • The game code doesn't static link with the plug-in library. 이 오류는 모든 Unreal 게임과 대부분의 자체 제작 엔진을 사용하는 게임에 일어날 수 있습니다.

  • The game cannot find the plug-in dynamic library (DLL or SO). 이 오류는 모든 Unity 게임과 일부 자체 제작 엔진을 사용하는 게임에 일어날 수 있습니다.

  • The game cannot load the plug-in dynamic library (DLL or SO) because of a missing dependency. 관련 플러그인 설명서에서 필수 조건을 확인하세요.

  • SoundBank가 최신 버전이 아닙니다.

  • SoundBank가 손상되었을 때.

  • AK::SoundEngine::GetDeviceList가 Init.bnk에 의해 혹은 AK::SoundEngine::RegisterPlugin을 명시적으로 호출하여 로드되지 않은 플러그인에 대해 호출되었습니다.

권장 해결 단계:

  • 게임이 Wwise 라이브러리와 고정적으로 연결될 경우 게임에서 #include "ThePluginFactory.h"와 연결 옵션을 확인하세요. Integration Details - Plug-Ins(통합 상세 정보 - 플러그인)에서 더 자세한 읽어보세요.

  • 게임이 동적 연결 (예: Unity)을 사용할 경우 SoundBank를 재생성하고 게임을 빌드하세요.

  • 게임에서 SoundBank를 재생성하고 재패키화하세요 (Init.bnk 포함).

  • 싱크 플러그인의 경우 이 플러그인의 ShareSet가 하나 이상의 마스터 버스와 연결되어 있는지 확인하세요.
