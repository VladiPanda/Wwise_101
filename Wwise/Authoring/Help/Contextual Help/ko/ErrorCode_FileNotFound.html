File ... not found in path(s): ...

지정한 파일을 찾을 수 없습니다. 검색된 경로가 열거됩니다.

유력한 원인:

  • SoundBank 경로가 설정되지 않았거나 올바른 폴더로 설정되지 않았습니다.

  • 해당 파일이 테스트 콘솔에 올바르게 배포되지 않았습니다.

  • 해당 파일이 Wwise 프로젝트에서 생성되지 않았습니다.

  • 해당 파일이 배포 패키지나 번들에 패키징되지 않았습니다.

권장 해결 단계:

  • Wwise와 함께 제공된 기본 I/O 코드를 사용하는 경우 CAkDefaultIOHookBlocking::SetBasePath(혹은 CAkDefaultIOHookBlocking::AddBasePath)에 대한 호출이 적절한 폴더로 향하는지 확인하세요.

  • SoundBank나 WEM 파일이 제대로 생성되었는지 확인하세요.

  • SoundBank가 올바르게 패키징되어 대상으로 배포되었는지 확인하세요.

  • Unity를 사용하는 경우 게임을 패키징하기 전에 Unity 프로젝트의 StreamedAssets 폴더에 파일이 있는지 확인하세요.

  • 맞춤 I/O 연결을 사용했을 경우 맞춤 I/O 연결 코드를 디버그하세요.
