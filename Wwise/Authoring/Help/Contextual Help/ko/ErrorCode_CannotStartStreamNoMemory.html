Not enough memory in I/O pool to start stream.

스트리밍용 파일의 헤더를 읽기에 I/O 풀의 메모리 양이 부족합니다. 사운드가 제거됩니다.

권장 해결 단계:

  • I/O 풀의 메모리 양을 늘이세요. 초기화 단계에서 Streaming Manager를 초기화하기 전에 AkDeviceSettings::uIOMemorySize를 설정하세요.

  • Wwise 프로파일러를 연결하여 Streaming 탭을 확인해 주세요. 스트리밍용 보이스의 수가 비정상적으로 많지는 않은지 확인해 주세요.

  • 가상 보이스와 제한을 사용하는 동시에 스트리밍하는 사운드의 개수를 줄이세요.
