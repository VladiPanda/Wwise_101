Source file is of different format than expected.

이 오류는 스트림된 파일이나 외장 음원이 연결된 구조적 SoundBank나 AK::SoundEngine::PostEvent() 호출에서 기록된 동일한 코덱을 사용하지 않을 경우 일어납니다.

유력한 원인:

  • 프로젝트의 Conversion Settings에서 코덱이 최근에 변경되었습니다.

  • External Source의 경우 AK::SoundEngine::PostEvent() 호출을 위해 AkExternalSourceInfo 구조로 전달된 형식에서 지정된 코덱이 실제 파일 형식과 대응하지 않습니다.

권장 해결 단계:

  • 모든 SoundBank와 WEM 파일 (스트리밍용 파일)이 생성되어 제대로 대상 콘솔에 배포되었는지 확인하세요.

  • External Source를 사용할 경우 WSOURCES 파일에서 지정된 Conversion Settings에 사용된 코덱을 검토하세요.
