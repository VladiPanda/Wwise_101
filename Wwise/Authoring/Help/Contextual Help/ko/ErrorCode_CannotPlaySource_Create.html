Failed creating source.

음원을 시작할 수 없었음을 나타내는 일반 오류입니다.

유력한 원인:

  • 음원이 등록되지 않은 플러그인을 사용하고 있습니다. 이 오류는 플러그인 오류가 발생한 후 일어날 가능성이 큽니다.

  • 음원을 생성하는 데에 필요한 메모리가 충분하지 않을 때 (이 경우 out of memory 오류가 먼저 뜨게 됩니다).

  • 이 음원이 연결된 버스가 존재하지 않을 때. Init SoundBank가 최신이 아닐 수 있습니다.

  • 예상하지 못한 오류.

권장 해결 단계:

  • 로그에서 메모리 부족 오류를 확인하고 더 많은 메모리를 할당하여 해결하세요. “Memory allocation failed.”을(를) 봐주세요.

  • Init SoundBank가 이 음원을 담고 있는 SoundBank와 동기화되었는지 확인하세요. SoundBank를 재생성하고 다시 복사하세요.

  • Audiokinetic 지원팀에 연락해 주세요. 문제를 재현하는 도중 ProfilingSession.prof를 캡처링하는 것이 좋습니다. (더 자세한 사항은 ???를 참고해 주세요)
