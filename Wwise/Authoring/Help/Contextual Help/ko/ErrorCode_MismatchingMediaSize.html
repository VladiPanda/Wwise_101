Mismatching media size error for file ... Possibly caused by mismatching sound bank or WEM resources.

이 오류는 Zero Latency 옵션을 사용하는 사운드에 대해서만 발생됩니다. WEM의 크기가 WEM 파일을 참조하는 SoundBank가 생성되었을 때의 크기와 같지 않다는 것을 Wwise가 감지했음을 나타냅니다. 이는 일반적으로 SoundBank와 WEM 파일의 생성, 패키징 또는 배포에 문제가 있음을 나타냅니다. 이 과정은 일반적으로 게임 엔진에서 처리됩니다.

유력한 원인:

  • 게임 내 SoundBank 및 WEM 파일의 패키징 또는 배포 오류입니다. 파일 중 하나라도 복사되지 않으면 이 문제가 일어나게 됩니다.

  • 사운드가 두 SoundBank에 중복되어 Zero Latency의 양 또한 중복됩니다. 한 SoundBank만 재생성된 경우 이 문제가 일어날 수 있습니다.

  • 오래된 SoundBank가 게임에 로드되었습니다.

  • 이 스트림을 읽을 때 I/O 연결에서 문제가 생겼습니다. 이 문제는 보통 맞춤화된 I/O 연결 사용과 관련됩니다.

  • 내부적인 버그.

권장 해결 단계:

  • SoundBank를 재생성하세요.

  • 오디오 파일 캐시를 제거하고 WEM 파일을 다시 변환하세요. 캐시 파일 제거는 File 메뉴에서 실행하거나 .cache 폴더에서 특정 WEM 파일을 직접 삭제함으로써 실행할 수 있습니다.

  • 테스트 콘솔에서 모든 .bnk와 .wem 파일을 삭제하세요.

  • Conversion Settings에서 다른 코덱을 사용하세요.

  • 이 문제가 지속되면 Audiokinetic 지원팀에 연락해주세요.
