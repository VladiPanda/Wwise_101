Cannot open file ... in path(s): ...

지정된 파일을 찾았지만 열거된 경로에서 접근할 수 없습니다. 이 오류는 일반적으로 파일 권한 오류이지만 다른 OS에서는 이 메시지로 이어지는 다른 오류를 제공할 수 있습니다.

해당 파일을 찾을 수 없으면 “File ... not found in path(s): ...” 오류가 대신 출력됩니다.

유력한 원인:

  • 파일 권한이 현재 사용자의 Read 작업을 방지하고 있습니다.

  • 운영 체제에서 알 수 없는 오류를 리턴했습니다.

권장 해결 단계:

  • 지정된 파일에 대한 게임의 배포 폴더에서 파일 권한을 확인하세요.
