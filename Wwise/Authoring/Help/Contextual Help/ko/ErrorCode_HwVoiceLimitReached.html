Maximum number of hardware-accelerated voices reached. Voice will not play.

하드웨어로 가속되는 오디오 형식을 사용하는 보이스가 동시에 너무 많이 재생됩니다.

유력한 원인

  • 동일한 오디오 형식을 소프트웨어에서 디코딩하는 다른 플랫폼과 동일한 변환 설정을 사용하고 있습니다. 일부 오디오 형식은 특정 플랫폼에서 자동으로 하드웨어 가속을 사용하도록 선택할 수 있습니다.

  • You are using a large number of multichannel, hardware-accelerated sounds. On some platforms, the number of channels counts against the limit, which means you can decode more mono/stereo sources concurrently than 5.1 or 7.1 sources.

  • 가상 보이스 스레숄드가 너무 낮게 설정되었습니다.

권장 해결 단계:

  • 일부 사운드에 다른 변환 설정을 사용하세요. 각 플랫폼과 사용 사례에 따른 최적의 변환 설정에 대한 정보는 오디오 형식을 참조하세요.

  • 하드웨어로 가속되는 오디오 형식을 사용하는 사운드가 보다 적게 동시 재생되도록 제한하세요.

  • 하드웨어로 가속되는 사운드에 모노나 스트레오 음원을 사용하세요.

  • 가상 보이스 스레숄드를 높이세요.
