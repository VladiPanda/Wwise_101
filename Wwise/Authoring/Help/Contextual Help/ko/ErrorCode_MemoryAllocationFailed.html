Memory allocation failed.

메모리 공간이 부족하여 메모리 할당에 실패했습니다 Wwise가 사용할 수 있는 최대 메모리는 초기화 시 AkMemSettings::uMemAllocationSizeLimit를 설정하여 코드에서 정의됩니다. Advanced Profiler의 Memory 탭에는 200ms마다 가져온 메모리의 현재 상태에 대한 스냅샷이 표시됩니다. 그렇기 때문에 상응하는 시간에 오류 사항이 보고되었음에도 불구하고 충분한 메모리가 있다고 나타날 수도 있습니다.

권장 해결 단계:

  • 사용하지 않는 SoundBank를 언로드하거나 SoundBank를 더 작은 파일로 분할하여 메모리를 덜 사용하세요. SoundBank는 보통 메모리를 가장 많이 사용하는 요소입니다.

  • 보다 적은 수의 Game Object를 등록 해제하여 메모리 사용을 줄이세요.

  • AkMemSettings::uMemAllocationSizeLimit를 늘리세요.
