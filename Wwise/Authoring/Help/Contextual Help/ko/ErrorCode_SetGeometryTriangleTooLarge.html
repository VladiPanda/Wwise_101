AK::SpatialAudio::SetGeometry - Triangle formed by vertices [v0, v1, v2] is too large.

AK::SpatialAudio::SetGeometry에 제공된 삼각형이 너무 큽니다.

유력한 원인:

  • 이 문제는 삼각형에서 두 개 이상의 꼭지점이 너무 멀리 떨어져 있어 특정 계산을 수행할 때 부동소수점이 오버플로우되어 발생합니다.

권장 해결 단계:

  • Profiler Settings(Alt-G)에서 API Calls을 활성화하여 게임이 수행하는 API 호출을 프로파일링하세요. 상황을 다시 시험하여 이 오류 이전에 해당 기능에 제공된 매개 변수 값을 확인하세요.

    오류 메시지에 있는 숫자는 너무 큰 삼각형들에 대한 Triangle 배열의 삼각형 색인과 Vertex 배열의 해당 꼭지점 색인을 표시합니다.
