AK::SpatialAudio::SetGeometry - More than two triangles are connected to the same edge.

AK::SpatialAudio::SetGeometry에 제공된 삼각형 중 하나가 제대로 구성되지 않았습니다. API는 두 개의 삼각형만 동일한 가장자리를 공유하도록 허용합니다.

권장 해결 단계:

  • Profiler Settings(Alt-G)에서 API Calls을 활성화하여 게임이 수행하는 API 호출을 프로파일링하세요. 상황을 다시 시험하여 이 오류 이전에 해당 기능에 제공된 매개 변수 값을 확인하세요.

    오류 문자열은 세 삼각형의 Triangle 배열에서 동일한 가장 자리를 잘못 되게 공유하고 있는 색인과 해당 가장 자리를 형성하는 Vertices 배열의 색인을 표시합니다.
