Density

인지 모달 밀도를 직접 변경합니다. 이 값이 낮을 수록 울리는 소리가 나타날 수 있으며, 이 값이 높을 수록 주파수가 더 평평해지며 잡음이 많은 잔향의 꼬리가 들리게 됩니다.

This property is directly proportional to the amount of memory and CPU used by the plug-in because higher modal density is achieved using longer delay lines. 이렇게 긴 지연 선은 큰 공간과 유사하기 때문에, 이 속성은 때때로 공간 크기의 인식을 조종하는 데에 쓰입니다. 이 속성은 ‘Room shape’와 ‘Quality’ 속성과 함께 긴밀하게 작동합니다.

Default value: 80
Range: 0 to 100
Units: %
