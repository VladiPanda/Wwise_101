Pre Delay

직접 소리와 고밀도 잔향 꼬리의 시작 부분 사이의 시간을 명시합니다. While pre delay supports exclusive RTPC, it should only be used for initialization, before playback. If an RTPC is used to change pre delay in real time during playback, the delay buffer will be continuously re-initialized, which will cause sudden silences or changes in the reverb signal.

단위: ms

Default value: 25
Range: 0 to 1000
