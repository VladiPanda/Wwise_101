Minimum System Audio Objects Requirement

サウンドエンジンがSystem Audio Objectを出すために、エンドポイントが対応しなくてはならない最小限のSystem Audio Object数を定義する。

このプロパティは、システムの3Dオーディオ用に一定容量を確保するために活用できる。例えば設定値を50にすると、50以上のAudio Objectに同時に対応できないプラットフォームでは、System Audio Objectの使用が実質的に無効となる。これにより、システムが対応できるオブジェクト数以上のオブジェクトの再生を試みたため、一部のオブジェクトが誤ったミキシングで処理されるような状況を回避できる。この状況が発生すると、再生するたびにオーディオの聞こえ方が異なってしまう。

そのためこの値は、ゲームで同時に再生されるSystem Audio Objectの最大数よりも、やや高めに設定することが推奨される。

Default value: 32
Range: 0 to 65535
