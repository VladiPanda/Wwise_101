Preserve System-Eligible Audio Objects

保留适宜系统音频对象。决定插件是否让带有 3D 空间化效果的 Audio Object 通过而不在 Main Mix 中对其进行混音。

  • When Audio Device supports it:插件查询总线管线末端的 Audio Device 来决定是对适宜 System Audio Object 进行混音还是让其通过。若 Audio Device 支持 System Audio Object，将保留 System Audio Object，否则将对其进行混音。
  • Always:保留适宜 System Audio Object，直至达到 System-Eligible Audio Object Limit，而无论 Audio Device 是否支持。
  • Never:始终在 Main Mix 中对适宜 System Audio Object 进行混音，而无论 Audio Device 是否支持。

Default value: When Audio Device supports it
