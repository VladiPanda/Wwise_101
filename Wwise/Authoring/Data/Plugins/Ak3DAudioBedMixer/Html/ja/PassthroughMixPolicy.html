Passthrough Mix

プラグインが、パススルー（スペーシャリゼーションなし）のミックスを表す追加のAudio Objectを生成するかどうかを決定する。

  • When Audio Device supports it:プラグインが、バスパイプラインの末端にあるAudio Deviceを参照し、パススルーミックスを生成するかどうかを判断する。Audio Deviceが対応する場合に限り、パススルーミックスを生成する。
  • Always:Audio Deviceの機能にかかわらず、パススルーミックスを生成する。
  • Never:Audio Deviceの機能にかかわらず、パススルーミックスを生成しない。

注釈:パススルーミックスを生成しない場合、パススルーミックスにミキシングされるAudio ObjectがMain Mixにミキシングされる。

Default value: When Audio Device supports it
